# blueberry_yield_prediction

## 项目介绍
本项目是本人的数据挖掘课程大作业。针对野生蓝莓产量预测构建机器学习模型，使用天池公开数据集，通过数据探索分析、特征工程、模型训练与评估，实现高精度产量预测。

## 数据集来源
数据集来源自
Obsie, E. Y., Qu, H., & Drummond, F. (2020).
Wild blueberry yield prediction using a combination of computer simulation and machine learning algorithms.
*Computers and Electronics in Agriculture*, 179, 105844.
https://doi.org/10.1016/j.compag.2020.105844

数据集在天池公开数据集中下载，https://tianchi.aliyun.com/dataset/211780
包含文件：
 - train.csv（训练集）
 - test.csv（测试集）

- 验证集 R²：0.8120，达到比较好的拟合度
- 预测结果分布合理
- 后续简单利用XGBoost、LightGBM等模型，发现 R²提升较少，大概提升1%左右
- 集成这几个模型，R²也提升很小0.8253，说明模型基本上达到了当前特征工程的上限

## 核心发现
### 1.变量相关性分析
通过相关性热力图分析变量间的关联关系，核心发现如下：
- 产量（yield）与坐果率（fruitset，0.89）、果实质量（fruitmass，0.83）、种子数（seeds，0.87）呈极强正相关，是产量预测的核心特征；
- 温度相关特征、降雨天数相关特征存在严重共线性，在特征工程中进行了降维处理；
- 克隆株大小（clonesize）与产量呈中等负相关（-0.38），为后续分组分析提供了依据。
### 2.克隆株大小分组分析
按克隆株大小（小/中/大）分组，分析坐果率、果实质量、种子数的分布差异：
- 随着克隆株增大，三个农艺性状的中位数均呈显著下降趋势，其中种子数的组间差异最为明显；
- 小克隆组性状数据离散度最高，大克隆组数据最集中，验证了克隆株大小对蓝莓农艺性状的显著影响。
### 3.降雨天数的影响
通过`AverageRainingDays`和`RainingDays`两个指标，分别进行「少雨/正常/多雨」分级和「极少雨/正常/极多雨」极端对比，核心结论如下：
- 随着降雨天数增加，蓝莓坐果率、果实质量、种子数均呈显著下降趋势；
- 极端多雨环境显著抑制蓝莓生长，适度少雨条件下蓝莓农艺性状表现最优；
- 两个降雨指标的分析结果高度一致，验证了结论的可靠性。
### 4.气温的影响
通过散点图矩阵分析平均最高温（`AverageOfUpperTRange`）与蓝莓生长指标的相关性：
- 平均最高温与3个生长指标的线性相关极弱，与现实情况不符，可能是温度特征选择不合理；
- 坐果率、果实质量、种子数三者间呈极强正相关（r>0.93，p<0.001），是产量预测的核心特征。
### 5.特征重要性
- 发现坐果率是影响产量的最重要的特征
